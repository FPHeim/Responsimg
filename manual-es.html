Manual de uso:

Crear las imágenes, agregarles los prefijos correspondientes a las distintas versiones con el formato "prefijo-nombre.ext", por ejemplo, "phone-ejemplo.jpg".

Importante: Las distintas versiones de una imagen deben estar en la misma carpeta.

Notas:
  • Se puede dejar sin prefijo ("nombre.ext") a la imagen original/por defecto.
  • El prefijo "none" no es válido por problemas con IE, Firefox y Opera.

Agregar el siguiente código a la hoja de estilos: 


body:after {
    visibility: hidden;
    display: none;
    width: 0;
    height: 0;
}

Completar la hoja de estilo con las asociaciones entre los distintos rangos de anchos de ventana y los prefijos de las imágenes correspondientes. 


@media (min-width: 1200px) {
    body:after {
        content: "prefijo"; /* Prefijo de la imagen */
    }
}

Ejemplo: 


@media (min-width: 481px) and (max-width: 767px) {
    body:after {
        content: "phone";
    }
}

Antes del </body> agregar el siguiente código: 


<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script src="js/responsimg.js" type="text/javascript"></script>  
<script type="text/javascript">
    $(function() {
        $.Responsimg({
            defaultPrefix: 'desktop' // Prefijo por defecto, util si el navegador no soporta media queries
        });
    }); 
</script>
